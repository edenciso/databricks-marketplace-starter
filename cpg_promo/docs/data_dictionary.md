# Data Dictionary — cpg_promo

- **features.sales_store_sku_day**: Daily store×SKU sales (no shopper data).
- **features.price_history**: List & promo prices.
- **features.promo_calendar**: Campaign windows and mechanics.
- **features.inventory_position**: On-hand and lead-time features.
- **features.events_calendar**: Holidays/sports/events.
- **features.weather_dma**: DMA-level weather.
- **features.creatives_meta**: Optional creative attributes/embeddings.
