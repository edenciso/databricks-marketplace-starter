# Quickstart — cpg_promo

1. Run `sql/01_create_tables.sql` to create Unity Catalog tables.
2. Run `notebooks/05_load_samples.py` to load synthetic sample data.
3. (Pro) Open `notebooks/10_promo_uplift_score.py` to score uplift.
4. (Studio) Open `notebooks/20_promo_planner_studio.py` and plan promotions/creatives.
